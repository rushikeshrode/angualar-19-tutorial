Pipes : Build-in & Custome
    Build-in Pipes :
  1. uppercase
  2. lowercase
  3. titlecase
  4. date:'fullDate'
  5. date:'short'
  6. currency:'INR'
  7. json
  8. slice:0:6

1: Build in Pipes :

  • uppercase, lowercase and titlecase

    • {{ 'hello angular' | uppercase }}

    {{ 'HELLO ANGULAR' | lowercase }}

    {{ 'angular is awesome' | titlecase }}

  • date:'short'/'fullDate'

    • Today is {{ today | date:'fullDate' }}

    Short: {{ today | date:'short' }}

  • percent

    • {{ 0.85 | percent }}

  • currency:'INR' default is USD

    • {{ 2500 | currency }}

    {{ 2500 | currency:'INR':'symbol' }}

    {{ 2500 | currency:'INR':'code' }}

    {{ 2500 | currency:'INR':'symbol-narrow' }}

  • json

    • {{ user | json }}

  • slice:0:6

    • {{ 'AngularRocks' | slice:0:6 }}

    2: Custome Pipe with Arguments

    >> cmd : ng generate pipe reverse
  • reverse custom-pipe
    • ex. rushikesh

    1: Rushikesh Prakashrao Rode

    => {{ 'Rushikesh Prakashrao Rode' | reverse }}

    2: Yash Golawar

    => {{ 'Yash Golawar' | reverse }}

  • cube custom-pipe

    • ex. 3 : 3*3*3=27

    => {{ 3 | cube }}

    ex. 9 : 9*9*9=81

    => {{ 9 | cube }}

  • powerTo : If arg not passed, It will power to 2.

    • Without argument : ex. 4^2

    {{ 4 | powerTo }}

    With argument : ex. 4^6

    {{ 4 | powerTo:6 }}
